# Screenshots & demo media

Add release visuals here for the **README**, **GitHub Release**, and **GitHub Pages** landing (`docs/index.html`).

## Expected files

| File | Suggested content |
|------|-------------------|
| `screenshot-chat.png` | Main chat — sidebar, model selector, thread |
| `screenshot-settings.png` | Settings modal (Providers or General) |
| `screenshot-library.png` | Library attachments table |
| `screenshot-search.png` | Ctrl+K search modal |
| `screenshot-dark.png` | Dark theme overview |
| `demo.gif` | Short screen recording (~5–10 s) — send message, stream reply |

Recommended width: **1280px** or **1440px** PNG. Keep GIF under **5 MB** for GitHub.

## Used by

- `README.md` → `docs/screenshots/…`
- `docs/index.html` → `./screenshots/…`
- GitHub Release body (drag-and-drop attachments)

Legacy copies may still exist under `assets/screenshots/`; prefer this folder for v2.5.0+.

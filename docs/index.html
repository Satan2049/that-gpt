Product overview

Desktop chat, redesigned — credentials stay in Rust

ThatGPT pairs a ChatGPT-inspired React UI with a Rust + Tauri backend. Markdown replies, agent tooling, multimodal attachments, and a responsive drawer layout — while conversations and presets persist as JSON on disk.

Download v2.0.0 GitHub
Credentials isolated AI_API_KEY is read only by the Rust backend, never exposed to the UI bundle.
No database layer Threads and presets are plain JSON in the app data directory for simple backup and inspection.
Agent-ready Tool calling, stop generation, regenerate, and rich attachments — images, audio, text, PDF.

Capabilities

What’s new in v2

Markdown & code

GFM rendering with syntax highlighting. Copy full messages or individual code blocks.

Agent workflow

Tool-calling loop, stop in-flight streams, and regenerate the last assistant reply.

Multimodal input

Images, audio, text files, and PDFs with optional vision and transcription models.

Responsive shell

Drawer navigation on narrow windows; full sidebar layout on desktop widths.

Design refresh

Design tokens, Inter typography, elevated composer, and polished dark mode.

Conversation tools

Rename threads, debounced sidebar search, export Markdown or JSON.

Prompt presets

Reusable system prompts with model and temperature per conversation.

In-app settings

API key, base URL, models, timeout, and retries — no manual file editing required.

Native desktop

Tauri 2 + Rust on Windows. Portable EXE or NSIS installer from GitHub Releases.

System design

Desktop shell layout

WebView (React) → Rust (Tauri) → Provider API

The UI calls Rust via Tauri invoke. The backend is the single integration point for AI calls and file I/O.

client/
React · Vite · Zustand → UI, shells, chat & preset forms
src-tauri/
Rust · Tauri → commands, validation, AI client, repositories

Installation

Quick start

  1. Option A Download a release 
    # GitHub Releases v2.0.0 → portable .exe or setup installer
# Verify SHA256.txt from the same release
  2. Option B Build from source 
    git clone https://github.com/Satan2049/that-gpt.git
cd that-gpt
npm install
npm run build
  3. Configure API access 
    # Open Settings in the app, or edit:
# %APPDATA%\com.thatgpt.desktop\.env
#
# AI_API_KEY, AI_BASE_URL, AI_MODEL, …
  4. Develop Run with hot reload 
    npm run dev
npm run clean   # remove build artifacts when needed
Security: ThatGPT is a single-user local desktop app. Report vulnerabilities via SECURITY.md.
